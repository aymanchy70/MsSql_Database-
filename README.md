## 📌 Project Overview

The **Digital Marketing Agency Management System (DMAMS)** is a comprehensive, SQL-based database solution designed to manage, track, and automate the core operations of a digital marketing agency. Built using **Microsoft SQL Server (MS SQL)**, this project serves as a robust enterprise management tool and a complete implementation of advanced relational database design, data manipulation, and optimization techniques.

The primary goal of DMAMS is to centralize agency workflows—seamlessly linking clients, active marketing campaigns, employee allocations, finances, and performance insights into a unified database structure.


## ⚡ Features & Benefits

* **🗃️ Centralized Architecture:** Manages clients, campaigns, employees, tasks, and financial records within a single, cohesive database schema.
* **🛠️ Full SQL Stack Implementation:** Features a practical application of advanced SQL operations, including complex Joins, Subqueries, Views, Stored Procedures, Triggers, Transactions, and Indexing.
* **📊 Campaign & Task Tracking:** Monitors project timelines, campaign performance metrics, budgets, and specific employee task assignments.
* **💰 Financial Integrity:** Accurately logs invoices, client payments, operational expenses, and employee salaries with built-in transactional security.
* **📈 Strategic Reporting:** Generates deep business insights through optimized queries to help agency stakeholders make data-driven decisions.


## 🗺️ Database Schema Summary

| Table Name | Purpose |

| **User** | Manages system authentication and monitors overall database/system activities. |
| **Clients** | Stores essential client profiles, including company info, contact details, and account status. |
| **Employees** | Contains internal team details, including roles, departmental assignments, and base salaries. |
| **Payment** | Securely records all incoming revenues and invoices received from agency clients. |
| **Expenses** | Tracks all outgoing operational costs required to keep the agency running day-to-day. |
| **Feedback** | Collects and logs quality assurance feedback from both system users and external clients. |
| **Service** | Catalogues available agency offerings across offline (e.g., photoshoots, video shoots) and online sectors (e.g., ad campaigns, digital product sales). |
| **Performance** | Monitors and evaluates employee Key Performance Indicators (KPIs) and productivity metrics. |


## 🛠️ Tech Stack & Key SQL Concepts Applied

* **Database Engine:** Microsoft SQL Server (MS SQL)
* **DDL & DML:** Structured table creation, relational constraints (Primary Keys, Foreign Keys), and data manipulation.
* **Advanced Programmability:**
* **Stored Procedures:** For automated, reusable business logic (e.g., processing payments, calculating salaries).
* **Triggers:** To enforce business rules and automate audit logging or state updates.
* **Views:** For securing sensitive data and simplifying complex multi-table reporting queries.
* **Transactions (ACID):** To guarantee financial data integrity during multi-step updates.
* **Indexes:** Implemented on high-frequency search columns to optimize query performance and scalability.
